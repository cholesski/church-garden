A project for Computer graphics course. The scene depicts a church garden at night.
Controls :
A,S,D,W - camera movement
B - turn bloom effect on/off
U - increase exposure
I - decrease exposure
F - flashlight on/off

Group A - cubemaps (skybox)
Group B - HDR, bloom (moon, flowers, lamp)
